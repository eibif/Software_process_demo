# 智考云在线考试平台原型

当前目录提供一套纯前端静态原型，用于展示在线考试平台中 1.2 用户管理模块的 Demo 实现。

## 目录结构

```text
index.html            页面入口
app.js                启动脚本
styles.css            样式聚合入口
styles/               分模块样式
scripts/              分模块脚本
README.md             原型说明
TEAM_GUIDE.md         组内扩展说明
```

## 已包含页面

- 登录
- 注册
- 重置密码
- 管理员首页
- 教师首页
- 用户列表
- 用户详情
- 用户新增 / 编辑
- 题库列表
- 试题详情
- 试题新增 / 编辑
- 个人中心
- 无权限提示页
- 考试管理（教师）
- 我的考试（学生）
- 考试详情（教师 / 学生）
- 新建 / 编辑考试（教师）

## 当前角色与范围

- 学生：查看个人资料与账号状态。
- 教师：查看本人及所属学生账号，维护学生资料、状态与密码。
- 管理员：查看全平台账号，负责角色分配、用户范围与全局维护。

## 主要能力

- 学生 / 教师 / 管理员统一身份入口
- 基于角色的菜单切换与路由控制
- 教师端用户范围收口，只显示本人及所属学生
- 管理员端全量用户管理与单角色分配
- 用户搜索、筛选、详情、编辑、冻结、恢复、重置密码
- 题型管理、试题录入、批量导入（演示）、检索筛选、提交审核与管理员审核
- 考试管理：创建考试、分配考试、参加考试
- 个人信息维护
- 会话冲突拦截

## 运行方式

直接打开 `index.html` 即可。
